TimeMachine – Quick Guide

This light in‑extension guide replaces the old long user guide. Full, always up‑to‑date documentation now lives in the project README.

Core Workflow

  1. Open popup → Sign up or log in with email + password (JWT token stored locally).
  2. Browse normally → Extension auto tracks active tab time in the background.
  3. Open popup anytime to see: daily / weekly / monthly charts, categories, productivity score.
  4. Change a site’s category → updates immediately and syncs to backend.
  5. Download / generate report (PDF) or schedule automatic ones.
  6. Send feedback from settings (authenticated).

Key Features

  • Automatic time tracking per domain (local + cloud sync).
  • Categories with instant reclassification.
  • Productivity score (Work + Professional + partial Other).
  • Report generation & optional scheduling.
  • Multiple themes & basic Pomodoro focus timer.

Reports

Use the “Download Report” button for a PDF. Scheduling (daily / weekly / monthly) happens locally; when trigger fires it calls the backend to generate the PDF.

Authentication (Simplified)

Only email + password now. No verification codes, no device confirmation emails. If token expires you’ll be asked to log in again.

Troubleshooting

  • No data? Keep the browser active on real http/https tabs; internal pages aren’t tracked.
  • Session expired? Just log in again; local unsynced time will flush once authenticated.
  • Wrong categories? Reassign in the list; chart auto-refreshes.
